# Plataforma-de-gestion-de-reservas-sobre-Incus
 Infraestructura como código para un laboratorio académico distribuido basado en contenedores Incus. Incluye automatización con OpenTofu y Ansible, monitoreo con Prometheus/Grafana, red OVN y almacenamiento persistente. Despliega microservicios de reservas para validar conectividad, persistencia y observabilidad en hardware modesto.
